Collection vs Collections

Collection
	- Root Interface of Collection Framework
	- i.e.
		- List
		- Set
		- Queue

Collections:
	- Utility Class for Collection objects (java.util)
	- i.e.
		- sort()
		- reverse()
		- shuffle()
		- binarySearch()
		- disjoint()
		- ...
